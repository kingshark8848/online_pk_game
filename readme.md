# PK Hub

## Intro

This is a realtime double player game platform SPA (single page application). 

### Tech stacks

- Express Framework (Nodejs)
- Socket.io (websocket lib)
- Bootstrap
- Vue.js

## Install & Configuration
### local development env
```
node app.js
```
Then visit `localhost:3000`
